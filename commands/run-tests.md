# Run Tests

Generate or run tests for the specified code.

## What I'll Do

1. Identify the testing framework used in the project
2. Find or create test files for the target code
3. Write tests covering core behavior, edge cases, and error paths
4. Run the test suite and report results
5. Fix any failing tests

## How to Use

```
Write tests for the auth service
Run tests for the user module
Add edge case tests for the payment handler
```

## Test Structure

- Use numbered test descriptions (e.g., `it("1. handles valid input")`)
- Group related tests in describe blocks
- Cover: happy path, edge cases, error handling
- Follow patterns in `.contextkit/standards/testing.md`

## Standards Applied

- `.contextkit/standards/testing.md` — Testing patterns and requirements
- `.contextkit/standards/code-style.md` — Coding conventions
